## Usage

```tsx
import {
  ExtensionAvatar,
  ExtensionAvatarFallback,
  ExtensionAvatarImage,
} from "<path-to-ui-component>/extension-avatar";
```

```tsx
<ExtensionAvatar>
  <ExtensionAvatarImage
    src="https://github.com/akashaorg.png"
    alt="@akashaorg"
    showLoadingIndicator={true}
  >
    <ExtensionAvatarFallback />
  </ExtensionAvatarImage>
</ExtensionAvatar>
```

## Props

| Name          | Type                     | Description                  |
| ------------- | ------------------------ | ---------------------------- |
| extensionId   | `string`                 | Extension id.                |
| size          | `sm`, `md`, `lg` or `xl` | Component sizes.             |
| extensionType | `ExtensionType`          | Extension type.              |
| nsfw          | `boolean`                | Not safe for work.           |
| publicImgPath | `string`                 | Path to public image folder. |
