## Usage

```tsx
import { Stack } from "<path-to-ui-components>/stack";
```

```tsx
<Stack>
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Stack>
```

## Props

| Name           | Type                                                                 | Description                                                                                                                                                                                      |
| -------------- | -------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| justifyContent | `start`, `center`, `end`, `between`, `evenly`, `around` or `stretch` | Determines the alignment of flex items along the container's main axis. For more details on Tailwind utilities for each value, refer to the [doc](https://tailwindcss.com/docs/justify-content). |
| alignItems     | `start`, `center`, `end`, `baseline` or `stretch`                    | Defines the alignment of flex items along the container's cross axis. For more details on Tailwind utilities for each value, refer to the [doc](https://tailwindcss.com/docs/align-items).       |
| direction      | `row`, `rowReverse`, `column` or `columnReverse`                     | Determines the direction of flex items. For more details on Tailwind utilities for each value, refer to the [doc](https://tailwindcss.com/docs/flex-direction).                                  |
| divider        | `boolean`                                                            | Add borders between items. For detailed information about the associated Tailwind utilities, please refer to the [doc](https://tailwindcss.com/docs/border-width#between-children).              |

<Callout>

The **default** value of `direction` is `column`.

</Callout>
