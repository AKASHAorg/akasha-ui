<Step>Run the following command to install `button.tsx`</Step>

```bash
npx shadcn@latest add "https://akasha-ui.pages.dev/r/styles/default/button.json"
```

<Step>Add the following colors to your CSS file</Step>

The command above should install the colors for you. If not, copy and paste the following in your CSS file.

```css title="app/globals.css"
@layer base {
  :root {
    --primary-start: 72.4% 0.177 350.2;
    --primary-end: 58.7% 0.204 277.4;
  }

  .dark {
    --primary-start: 72.4% 0.177 350.2;
    --primary-end: 58.7% 0.204 277.4;
  }
}
```

Add the following object in the `theme.extend.colors` section of your `tailwind.config.js` file
this config enable button related style utilities like `from-primary-start`

```javascript title="tailwind.config.js"
// ...
primary: {
  start: 'hsl(var(--primary-start))',
  end: 'hsl(var(--primary-end))',
},
// ...
```

## Usage

```tsx
import { Button } from "<path-to-ui-components>/button";
```

```tsx
<Button variant="outline">Button</Button>
```

## Link

You can use the `link` variant to create a link button.

```tsx
import { Button } from "@/components/ui/button";
```

```tsx
<Button variant="link">Click here</Button>
```

Alternatively, you can set the `asChild` parameter and nest the anchor element.

```tsx
<Button asChild>
  <a href="/link">Link</a>
</Button>
```
