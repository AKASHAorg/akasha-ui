---

**AKASHA UI** is built on top of [shadcn/ui](https://ui.shadcn.com), and you can follow the official [installation](https://ui.shadcn.com/docs/installation) guide to get started.

For projects using Tailwind version 4 or later, refer to this [guide](https://ui.shadcn.com/docs/tailwind-v4) to set up the necessary dependencies.
